== Description ==

This addon for iThemes Exchange adds SKU support to all Product Types.

For more information visit: http://www.visser.com.au/exchange/

== Installation ==

1. Upload the folder 'exchange-addon-add-product-sku' to the '/wp-content/plugins/' directory
2. Activate 'iThemes Exchange - Add Product SKU' through the 'Plugins' menu in WordPress
3. Open the 'Addons' screen through the 'Exchange' menu in WordPress
4. If Add Product SKU is not enabled click Enable beside the 'Add Product SKU' entry in the Addons listing
